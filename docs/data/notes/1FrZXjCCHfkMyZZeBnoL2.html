Ch01 【序章】

  • 定義 Server: 對企業內部或網際網路上的用戶端(Client)提供各種服務的電腦。
  • 用戶端和伺服器之間的關係: 由伺服器和用戶端架構而成的系統就稱為「用戶端/伺服器系統(Client/Server System)，或稱為主從式架構」，是網路世界最常見的服務架構。
  • Server 會提供不同服務，所謂的伺服器其實就是一部安裝了「伺服器軟體」的電腦，不同的伺服器軟體提供不同的服務。
